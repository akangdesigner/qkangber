## 解決什麼問題

週報、月報靠人工從各系統複製貼上資料，每次要花 2–4 小時，還容易算錯。老闆想要即時數據，但人力跟不上。

## 自動化流程

1. **多源資料整合** — 自動從 GA4、廣告後台、電商平台、CRM 抓取數據
2. **自動計算** — KPI 指標運算、環比同比計算，無需人工處理
3. **視覺化報表** — 自動更新 Google Data Studio / Looker Studio 儀表板
4. **定時發送** — 每週一自動發送 Email 報告給指定收件人
5. **異常警示** — 數據超出閾值時立即通知（業績驟降、廣告費暴增等）

## 服務內容

- 報表需求訪談與設計
- 各數據源 API 串接
- n8n 排程工作流建置
- Google Sheets / Data Studio 模板客製
- 上線後 2 週調整
